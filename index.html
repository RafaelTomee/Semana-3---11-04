Sobre o site

Este site foi criado com o objetivo de praticar a criação de páginas em HTML utilizando tags que eu ainda não havia usado, como <header>, <footer>, <section>, <form>, <label>, <input>, e <textarea>.

Adicionei formulários de contato e login para praticar o uso dessas novas tags.

O que estou aprendendo

  • Uso da tag <header> para criar o cabeçalho da página
  • Uso da tag <footer> para criar o rodapé
  • Uso da tag <section> para organizar o conteúdo
  • Uso da tag <form> para criar formulários de entrada de dados
  • Uso da tag <label> para associar rótulos aos campos de formulário
  • Uso da tag <input> para campos de texto e senha
  • Uso da tag <textarea> para campos de mensagem multilinha
  • Como estruturar um site somente com HTML

Este é um dos meus primeiros sites e está me ajudando a fixar os conceitos básicos de HTML, incluindo formulários e campos interativos.
