Sobre o site

Este site foi criado com o objetivo de praticar a criação de páginas em HTML utilizando tags que eu ainda não havia usado, como <header>, <footer> e <section>.

É um exercício simples, mas muito útil para entender a estrutura básica de uma página web.

O que estou aprendendo

  • Uso da tag <header> para criar o cabeçalho da página
  • Uso da tag <footer> para criar o rodapé
  • Uso da tag <section> para organizar o conteúdo
  • Como estruturar um site somente com HTML

Este é um dos meus primeiros sites e está me ajudando a fixar os conceitos básicos.
